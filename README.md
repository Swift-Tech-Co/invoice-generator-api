Lightweight API for generating professional PDF invoices. Supports custom line items, tax rules, multi-currency formatting, logo upload, and payment terms. Returns a PDF binary or a hosted link.

Built for SaaS platforms that need invoice generation without a full billing system.

**Languages:** Node.js / TypeScript, Python
